## 📋 Prerequisites (install)

### 1️⃣ Install Termux
```bash
# Download and install Termux from F-Droid
# https://f-droid.org/en/packages/com.termux/
```

### 2️⃣ Update Termux
```bash
apt update -y
apt upgrade -y
```

### 3️⃣ Install Required Packages
```bash
# Install required tools
apt install wget curl proot tar git nano vim -y
```
